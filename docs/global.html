    leftpad(input, length, text) → {string}

    This funciton applies the padding on left side of the input using the given length and given text to pad The given text's length should not be more than 1

    input number | string the input number to apply the padding on - (can be of any length)
    length number 3 the lenght that you want the input to be of - (can be any positive integer number or natural number), @default - 3
    text number | string 0 the number or text that will be used to pad - (can be ony the lengh of 1), @default - 0
